# React Query 도입 전·후 지연 발생량 비교

## 1. 정의

- **지연 발생량**: 사용자 행동(페이지 진입, 전공/필터 선택, 상세 진입 등)에 따라 발생하는 **네트워크 요청 횟수** 및 그로 인한 **체감 대기 시간**.
- **비교 대상**: React Query 미사용(매 요청마다 `fetch`) vs 사용(캐시 + `staleTime`).

---

## 2. 현재 React Query 적용 구간

| 훅 | API | staleTime | 사용처 |
|----|-----|-----------|--------|
| `useRecommendations(major, limit)` | 전공별 추천 | 10분 | RecommendationPage |
| `useMajors()` | 전공 목록 | 10분 | RecommendationPage, MyPage, UserMenu |
| `usePopularMajors(limit)` | 인기 전공 | 10분 | RecommendationPage, MyPage |
| `useCerts(params)` | 자격증 목록 | 10분 | CertListPage |
| `useFilterOptions()` | 필터 옵션 | 10분 | CertListPage |
| `useCertDetail(qualId, token)` | 자격증 상세 | 1시간 | CertDetailPage |
| `useCertStats(qualId, year)` | 자격증 통계 | 1시간 | CertDetailPage |

---

## 3. 시나리오별 지연 발생량 차이

### 3-1. 전공별 추천: 같은 전공 2번 선택

| 구분 | React Query 없음 | React Query 있음 |
|------|------------------|------------------|
| 1차 선택 "컴퓨터공학" | 1회 요청 → 지연 1회 | 1회 요청 → 지연 1회 |
| 다른 페이지 갔다가 복귀 후 "컴퓨터공학" 재선택 | 1회 요청 → 지연 1회 | **캐시 적중 → 지연 0회** |
| **총 지연 발생** | **2회** | **1회** |
| **감소** | — | **1회 분 지연 제거 (약 50%)** |

### 3-2. 자격증 목록: 같은 필터로 목록 2번 진입

| 구분 | React Query 없음 | React Query 있음 |
|------|------------------|------------------|
| 1차 진입 (필터 A) | 목록 1회 + 필터옵션 1회 = 2회 | 2회 |
| 다른 페이지 갔다가 같은 필터로 재진입 | 목록 1회 + 필터옵션 1회 = 2회 | **캐시 적중 → 0회** |
| **총 지연 발생** | **4회** | **2회** |
| **감소** | — | **2회 분 지연 제거 (약 50%)** |

### 3-3. 자격증 상세: 같은 상세 2번 방문

| 구분 | React Query 없음 | React Query 있음 |
|------|------------------|------------------|
| 1차 상세 진입 (상세 + 통계) | 2회 | 2회 |
| 목록으로 돌아갔다가 같은 상세 재진입 | 2회 | **캐시 적중 → 0회** |
| **총 지연 발생** | **4회** | **2회** |
| **감소** | — | **2회 분 지연 제거 (약 50%)** |

### 3-4. 전공 목록: 여러 페이지에서 사용

- `useMajors()` 사용처: RecommendationPage, MyPage, UserMenu.
- **React Query 없음**: 각 페이지/메뉴 마운트마다 전공 목록 API 1회 → 진입할 때마다 지연 1회.
- **React Query 있음**: 첫 요청 1회 후 10분간 캐시 → 다른 페이지 갔다 와도 전공 목록 재요청 0회.
- **감소**: 동일 세션 내 전공 목록 중복 요청 제거 → **해당 요청 수만큼 지연 발생량 감소**.

---

## 4. 요약 (한 줄)

- **재방문·동일 파라미터 재요청 시** 캐시 적중으로 **해당 요청의 네트워크 지연이 0**이 되어,  
  **전공별 추천·자격증 목록·상세·전공 목록 등에서 반복 요청 분 지연이 약 50% 수준으로 줄어든다.**

(백엔드 응답 시간을 R(ms)이라 하면, 캐시 적중 1회당 **R ms만큼** 사용자 체감 지연이 감소한다.)
